Given a fence with n posts and k colors,
make an algorithm to find out the number of ways of painting the fence
so that not more than two consecutive posts have the same colors.

Input:
n = post count
k = color count

Example:
Input: n = 3, k = 2
Output: 6
Explanation: The six ways are:
            w1 w2 w3
            1 2 1
            1 2 2
            2 1 1
            2 1 2
            2 2 1
            2 2 2
